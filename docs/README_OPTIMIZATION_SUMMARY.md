## 🔧 技术细节

### GIF 转换命令（完整版）
```bash
# APK 演示 GIF（完整 6分16秒）
ffmpeg -i "演示视频/AI分析apk演示.mkv" \
  -vf "fps=10,scale=1000:-1:flags=lanczos,split[s0][s1];[s0]palettegen=max_colors=128[p];[s1][p]paletteuse=dither=bayer:bayer_scale=5" \
  -loop 0 "assets/demo-apk.gif" -y

# EXE 演示 GIF（完整 2分44秒）
ffmpeg -i "演示视频/AI分析exe演示.mkv" \
  -vf "fps=10,scale=1000:-1:flags=lanczos,split[s0][s1];[s0]palettegen=max_colors=128[p];[s1][p]paletteuse=dither=bayer:bayer_scale=5" \
  -loop 0 "assets/demo-exe.gif" -y
```

### 优化策略
- **两步调色板生成**：先生成 128 色最优调色板，再应用到帧序列
- **Lanczos 缩放**：高质量图像缩放算法
- **Bayer 抖动**：平滑色彩过渡，减少色带
- **完整时长**：保留完整分析流程，不截断
