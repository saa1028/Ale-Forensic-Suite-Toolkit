# Notice

This repository is a fork and derivative maintenance workspace based on:

- Upstream: https://github.com/makoto56/penetration-suite-toolkit
- Original author badge shown by upstream: Makoto56

The purpose of this fork is to maintain a forensic-oriented toolkit virtual machine for digital forensics, incident response, traffic analysis, memory analysis, mobile forensics, malware triage, and CTF Forensics.

## License note

At the time this fork was created, the upstream repository did not declare an explicit open-source license. Without an explicit license, redistribution and derivative rights may be limited by default copyright rules.

For this reason, this fork aims to:

- Keep attribution and upstream links visible.
- Prefer tools with clear licenses and official distribution channels.
- Avoid adding unauthorized commercial binaries or unclear third-party packages.
- Document the source, license, and intended use of newly added tools whenever possible.

If you are the upstream maintainer or a rights holder and want attribution corrected or content removed, please open an issue.
